What technologies are used for this project?
This project is built with:

Vite
TypeScript
React
shadcn-ui
Tailwind CSS

client portal / freelancer dashboard web app.  
Features:  
- Two roles: Client and Freelancer.  
- Clients can post jobs, track progress, view deliverables, approve/decline work.  
- Freelancers can see assigned jobs, upload work files, submit updates, message client.  
- Messaging / chat inside the portal.  
- Notifications for status changes (e.g. “work submitted”, “approved”, “revision requested”).  
- Project timeline / milestone view.  
- Payment invoicing: generate invoices (mock or real payment gateway).  
- Profile pages for both roles (skills, portfolio, ratings).  
- Dashboard overview: active projects, pending tasks, earnings.  
- File upload / version history.
